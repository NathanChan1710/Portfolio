Analyse Réseaux et Infrastructure pour la SNCF
Description du projet
SNCF Réseau et Voyageurs Datavisualisation
Janvier 2026

L’analyse s’inscrit dans la mission centrale de SNCF Réseau et SNCF Voyageurs : garantir un réseau fiable, disponible et sécurisé pour des millions d’usagers. Le fonctionnement quotidien dépend d’un parc immense d’équipements dont la conception, le déploiement et la maintenance exigent une surveillance continue. Chaque défaut technique, chaque incident détecté par les trains constitue une source précieuse de retour d’expérience pour anticiper les risques et améliorer la performance du réseau.

Méthodologie
  • Préparation des données et nettoyage
  • Analyse descriptive des données sur Power BI
  • Conception du Power BI pour respecter la charte graphique de la SNCF
  • Respect des deadline, présentation du projet en amphithéâtre
Technologies utilisées
R Jointure Data Open Source SNCF Power BI
Résultats & Conclusions
  • Ligne entre Paris/Dijon/Lyon : nombreux événements trains
  • Sur les tranches de tensions y en a une dizaine qui ont des anormales + de 10.000KW d’écarts environ
  • Certains Evènement n’arrive qu'à l’arret
  • Zone de Paris avec le plus d’écart de tension mais aussi le plus d’incidents liés à la tension en mars 2013

Renault Financial Analysis
Description du projet
Finance Data Visualization
Janvier 2026

Analyse approfondie des données boursières de l'action Renault (composante du CAC40). Ce projet combine l'automatisation Python pour la récupération de données en direct avec la visualisation avancée dans PowerBI, permettant une analyse financière complète et actualisée.

Méthodologie
  • Récupération automatique des données boursières via le package Python Yfinance
  • Script Python pour charger les données en direct et les transformer
  • Construction d'un dashboard PowerBI avec KPIs financiers clés
  • Analyse technique avec candlestick et moyennes mobiles
Technologies utilisées
PowerBI Python Yfinance Données boursières
Résultats & Conclusions

Événement clé analysé : Abaissement des prévisions 2025 par Renault

  • Marge opérationnelle : Passage de 7% à 6,5% pour 2025
  • Leadership : Nomination d'un CEO intérimaire ajoutant de l'incertitude
  • Impact boursier : Analyse des mouvements de prix et volatilité avant/après annonce
  • KPIs du dashboard : Variation de cours, volumes d'échange, moyennes mobiles (20j, 50j, 200j)
