Renault Financial Analysis
Description du projet
Finance Data Visualization
Janvier 2026

Analyse approfondie des données boursières de l'action Renault (composante du CAC40). Ce projet combine l'automatisation Python pour la récupération de données en direct avec la visualisation avancée dans PowerBI, permettant une analyse financière complète et actualisée.

Méthodologie
  • Récupération automatique des données boursières via le package Python Yfinance
  • Script Python pour charger les données en direct et les transformer
  • Construction d'un dashboard PowerBI avec KPIs financiers clés
  • Analyse technique avec candlestick et moyennes mobiles
Technologies utilisées
PowerBI Python Yfinance Données boursières
Résultats & Conclusions

Événement clé analysé : Abaissement des prévisions 2025 par Renault

  • Marge opérationnelle : Passage de 7% à 6,5% pour 2025
  • Leadership : Nomination d'un CEO intérimaire ajoutant de l'incertitude
  • Impact boursier : Analyse des mouvements de prix et volatilité avant/après annonce
  • KPIs du dashboard : Variation de cours, volumes d'échange, moyennes mobiles (20j, 50j, 200j)
